# 3주차: JavaScript 기초

## 학습 목표

- 변수 선언 방식(let, const) 이해
- 데이터 타입과 형변환
- 함수, 조건문, 반복문 작성

## 핵심 개념

| 개념 | 설명 |
|------|------|
| `let` / `const` | 블록 스코프 변수 선언 (var 사용 지양) |
| 데이터 타입 | string, number, boolean, null, undefined, object |
| 함수 | 선언식 vs 표현식 vs 화살표 함수 |
| 조건문 | `if / else`, 삼항 연산자 |
| 반복문 | `for`, `while`, `forEach` |
