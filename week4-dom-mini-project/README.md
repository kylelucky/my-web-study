# 4주차: DOM 조작 & 미니 프로젝트

## 학습 목표

- DOM 요소 선택 및 동적 조작
- Event Listener로 사용자 인터랙션 처리
- Fetch API로 외부 데이터 불러오기

## 핵심 개념

| 개념 | 설명 |
|------|------|
| DOM 선택 | `querySelector`, `getElementById` |
| DOM 조작 | `innerHTML`, `classList`, `setAttribute` |
| Event Listener | `addEventListener('click', handler)` |
| Fetch API | `fetch(url).then(res => res.json())` |
