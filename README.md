- 웹의 기본 구조(HTML), 스타일(CSS), 동작(JavaScript)을 단계적으로 학습
- 단순 구현을 넘어 **문제 해결 능력**과 **디버깅 과정**을 기록

## 🛠 Troubleshooting 기록 형식

학습 중 발생한 문제와 해결 과정을 각 주차 README에 기록합니다.

```
[문제] 어떤 문제가 발생했는가
[원인] 왜 발생했는가
[해결] 어떻게 고쳤는가, 그리고 왜 그 방법이 맞는가
```

> 단순 해결이 아니라 **"왜 문제가 발생했는지"** 까지 분석하는 것을 목표로 합니다.
