각 단계에서의 **문제 해결 과정(Troubleshooting)**까지 기록하는 저장소입니다.

## 📌 프로젝트 목표

* 웹의 기본 구조(HTML), 스타일(CSS), 동작(JavaScript)을 단계적으로 학습
* 단순 구현을 넘어 **문제 해결 능력**과 **디버깅 과정**을 기록

| 기술         | 역할           | 설명                              |

### ✅ 1주차: HTML & Web 기초

## 🛠 Troubleshooting 기록

학습 중 발생한 문제와 해결 과정을 정리합니다.

**예시 형식**

```
[문제]
- 버튼 클릭 이벤트가 동작하지 않음

[원인]
- querySelector 선택자가 잘못됨

[해결]
- 올바른 class 선택자로 수정
```

👉 단순 해결이 아니라
**"왜 문제가 발생했는지"**까지 분석하는 것을 목표로 합니다.
