*HTML (Structure): 건물의 설계도이자 뼈대입니다. 제목, 문단, 이미지 위치를 결정합니다.

*CSS (Presentation): 건물의 인테리어입니다. 색상, 폰트, 배치(Layout)를 아름답게 꾸밉니다.

*JavaScript (Behavior): 건물의 가전제품이나 승강기입니다. 클릭 시 메뉴가 열리거나 데이터가 변하는 동적인 기능을 만듭니다.

위 3가지를 공부해보고자 합니다. 공부하며 발생한 문제에 대한 troubleshooting 과정 또한 기록하고자 합니다.
